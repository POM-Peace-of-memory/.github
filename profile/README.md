## 👥 About Us

**(POM) Peace of Memory**는 사용자의 소중한 기억들을 모아 그룹별로 저장하고 관리하는 서비스를 제공합니다. <br/>
코드잇 부스트 동아리의 파워 부스트 토이 프로젝트로 시작된 우리는 사람들이 추억을 쉽게 관리하고 공유할 수 있도록 돕는 것을 목표로 하고 있습니다.
